# Gemma 4 Skills for Google AI Edge Gallery

A growing collection of community skills for **Gemma 4** running on
[Google AI Edge Gallery](https://github.com/google-ai-edge/gallery)
(Android & iOS).

All skills run **100% on-device** — no data leaves your phone.

---

## 🧠 What is this?

Google AI Edge Gallery lets you run Gemma 4 entirely offline on your phone.
**Agent Skills** are modular packages that extend what the model can do —
think of them as plugins that give Gemma 4 new abilities like fetching news,
generating QR codes, or running custom logic.

This repository hosts skills focused on **business, productivity, and
LATAM-specific use cases**, built for entrepreneurs and operators who
want fast, private AI on the edge.

## 🚀 How to install a skill

Open **Google AI Edge Gallery** → Agent Skills → tap the **Skills** chip →
tap **(+)** → **Load skill from URL**, and paste the skill folder URL.
